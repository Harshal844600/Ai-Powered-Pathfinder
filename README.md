## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun
- OpenAI API Key

### Installation

1. **Clone the repository**
   
```
bash
   git clone https://github.com/Harshal844600/Ai-Powered-Pathfinder.git
   cd Ai-Powered-Pathfinder
   
```

2. **Setup Frontend**
   
```
bash
   cd "Ai Roadmap Gen"
   npm install
   # or
   bun install
   
```

3. **Setup Backend**
   
```
bash
   cd ../Ai-Roadmap-Backend
   npm install
   
```

4. **Environment Variables**

   Create a `.env` file in `Ai-Roadmap-Backend/`:
   
```
env
   OPENAI_API_KEY=your_openai_api_key_here
   PORT=3000
   
```

5. **Start Development Servers**

   Backend:
   
```
bash
   cd Ai-Roadmap-Backend
   npm start
   
```
   
   Frontend:
   
```
bash
   cd "Ai Roadmap Gen"
   npm run dev
   
```

   The frontend will be available at `http://localhost:5173`
